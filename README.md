## `injectsync@automation $` **`show --about`**

InjectSync is the official automation and infrastructure account operating across the InjectCore ecosystem.

This account manages deployment pipelines, repository synchronization, infrastructure orchestration, runtime services, monitoring workflows, and internal engineering automation.

---

## `injectsync@automation $` **`show --services`**

- CI/CD workflow execution and orchestration
- Infrastructure provisioning and synchronization
- Deployment automation and runtime operations
- Repository management and integration workflows
- Internal tooling and operational services
- Monitoring, observability, and service automation
- Security-oriented operational processes
- Cross-platform engineering support systems
